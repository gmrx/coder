# Coder

AI-ассистент для VS Code с агентным циклом и поддержкой любого LLM API.

**Автор:** gmrx
**Лицензия:** Доступ только по личному соглашению с автором.

## Установка в режиме разработки

1. Установите зависимости:

```bash
npm install
```

2. Соберите расширение:

```bash
npm run build
```

3. Откройте эту папку в VS Code.

4. Нажмите `F5` (Run Extension) — откроется новое окно VS Code с загруженным расширением.

## Настройка

В VS Code зайдите в:

- **Settings** → найдите `AI Assistant`.

Доступные настройки:

- `aiAssistant.apiBaseUrl` — базовый URL API (endpoint вашей модели).
- `aiAssistant.apiKey` — ваш API-ключ (не коммитьте его в git!).
- `aiAssistant.model` — имя модели (например, `gpt-4.1-mini` или другое).

## Использование

1. Выделите текст в редакторе (опционально) — он подставится как запрос.
2. Вызовите команду:
   - через палитру команд `Ctrl+Shift+P` / `Cmd+Shift+P` → **AI Assistant: Спросить модель**;
   - или привяжите хоткей к команде `ai-assistant.ask`.
3. Введите или отредактируйте запрос.
4. Ответ откроется в новом markdown-документе.

## Примечания

- Расширение использует обычный HTTP `POST` с JSON-форматом, совместимым с OpenAI Chat Completions API.
- Вы можете подставить свой endpoint, если он совместим по формату или сделать небольшой форк и поменять структуру запроса/ответа.
